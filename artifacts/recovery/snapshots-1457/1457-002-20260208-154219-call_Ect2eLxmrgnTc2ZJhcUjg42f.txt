|---|---|

| `partial` | 1-2 required layers are missing tasks |
| `gap` | 3+ required layers are missing tasks |
| `at_risk` | Feature is `done` or `in_progress` but still has missing required layers |

- `not_required` = layer is not required for this feature type
- `missing` = required layer has no linked CSV task
- `done/in_progress/planned/partial (task IDs listed)` = layer has linked task coverage
