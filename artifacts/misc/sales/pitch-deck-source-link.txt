## Deliverables Required

### Primary Deck
- **Format**: 12-slide master deck
- **Outputs**: PDF + editable source (Figma/Canva)
- **Content**: Follow outline in `docs/sales/pitch-deck-outline.md`
- **Design**: Follow guidelines in `docs/company/brand/visual-identity.md`

### Persona Variants
Create 4 customized versions targeting:
1. **Founder/GM**: Emphasize business impact, ROI, predictable execution
2. **Sales Lead**: Focus on features, UX, time savings
3. **RevOps/Ops**: Deep dive on governance, validation, data quality
4. **Engineering Lead**: Technical stack, architecture, observability

### Supporting Materials
- Speaker notes for each slide
- Demo flow recommendations
- Objection handling guide
- Appendix slides (optional)

## Design Requirements

### Visual Style
- **Brand Attributes**: Modern, reliable, technical but approachable
- **Color Palette**: Use semantic tokens from `docs/company/brand/palette.tokens.json`
- **Typography**: Use tokens from `docs/company/brand/typography.tokens.json`
- **Spacing**: Use tokens from `docs/company/brand/spacing.tokens.json`

### Layout Guidelines
- Generous white space
- Left-align text for readability
- Maximum 3-4 bullet points per slide
- Visual hierarchy: headline → subheadline → body → CTA
- Slide template with consistent header/footer
- Logo placement (top left or bottom right)
- Slide numbers for navigation

### Content Guidelines
- Follow slide-by-slide outline in `pitch-deck-outline.md`
- Include persona-specific talking points
- Reference objection handling from `docs/company/go-to-market/objections.md`
- Use copy blocks from `docs/company/messaging/copy-blocks.md`
- CTAs from `docs/company/messaging/cta-library.md`

## Timeline

**Total**: ~2 weeks to final deck

1. **Design Draft**: 1 week
   - Create master deck following outline
   - Apply brand guidelines
   - Add visual elements and diagrams

2. **Internal Review**: 3-5 days
   - PM + Sales Team review
   - Gather feedback
   - Test with sample personas

3. **Revisions**: 2-3 days
   - Incorporate feedback
   - Refine visuals and messaging
   - Create persona variants

4. **Final Approval**: 1 day
   - Final sign-off
   - Export all formats
   - Distribute to sales team
