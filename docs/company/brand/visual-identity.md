# Visual Identity

This is an initial brand system for IntelliFlow CRM. Update tokens in:

- `docs/company/brand/palette.tokens.json`
- `docs/company/brand/typography.tokens.json`
- `docs/company/brand/spacing.tokens.json`

## Brand Attributes

- Modern
- Reliable
- Technical but approachable

## Color

Use the token files as the source of truth. Prefer semantic tokens (e.g.,
`color.brand.primary`) over hard-coded hex values.

## Typography

Default to a clean sans-serif for UI and documentation.

## Spacing

Use a consistent spacing scale from the spacing tokens file.
