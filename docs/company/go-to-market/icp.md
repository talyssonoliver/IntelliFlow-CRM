# ICP Definition

## Who We Serve

Teams that need a modern CRM with automation and governance, without adopting a
heavy enterprise suite.

## Firmographics

- SMB to mid-market
- B2B SaaS / services
- Distributed teams

## Technographics

- Uses GitHub + CI/CD
- Comfortable with modern web tooling
- Values observability (metrics/logs/traces)

## Must-Have Requirements

- Reliable automation with safeguards
- Clear data validation rules and auditability
- Fast UI + API performance
