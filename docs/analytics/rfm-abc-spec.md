# RFM + ABC Segmentation Spec

## Purpose

Define a segmentation approach to prioritize customers and outreach.

## RFM

- **Recency**: days since last activity
- **Frequency**: count of key events over a window
- **Monetary**: revenue or proxy value

## ABC (Pareto)

- A: top ~20% accounts contributing ~80% value
- B: middle tier
- C: long tail

## Outputs

- Segment labels per account
- Rules for moving between segments
- Audit fields and timestamps
